Shib-Faker
===========

BETA - Use at own risk
-----------------------

Fakes HTTP Headers that Shibboleth usually injects after intercepting requests

requires node and http-proxy from NPM

#### Use in the Terminal

	node shib-faker.js <app_host:app_port> <proxy host:proxy_port>
